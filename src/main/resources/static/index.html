QueryBoard/api
v1.0

A simple
query & discussion
REST API.

Create topics, raise queries, comment, and engage — all through clean HTTP endpoints.

Platform Rules
  • 01 Users can create categories for questions — e.g. Entertainment, Food, Sports, etc.
  • 02 Users can raise queries under any existing category/topic.
  • 03 Anyone can view all queries under any topic — no login required.
  • 04 Users with system access can comment, reply, and like queries.
  • 05 A query can only be edited or deleted by its original owner.
Authentication
⚡
Login is required to raise, edit, or delete queries.
No login needed to view queries (/view) or to like & comment on them.
Endpoints
GET /view public View all queries
POST /Query auth required Raise a new query

Send a JSON body with the following fields:

{
  "Topic":  "Entertainment",
  "Query":  "What are the best sci-fi movies of 2024?"
}
PUT /Query/:id auth required Edit your query (owner only)
DELETE /Query/:id auth required Delete your query (owner only)
POST /Query/:id/comment public Comment on a query
POST /Query/:id/like public Like a query
POST /category auth required Create a new category
