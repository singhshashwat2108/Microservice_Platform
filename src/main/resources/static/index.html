Platform Rules
1

Users can create categories for queries — e.g. Entertainment, Food, Sports.

2

Logged-in users can raise queries under any existing category.

3

Anyone can view all queries under any topic — no login required.

4

Anyone can comment and like queries without logging in.

5

Only the owner of a query can edit or delete it.

API Endpoints
GET /view View all queries public
GET /view?topic=... Filter by topic public
POST /auth/register Create an account public
POST /auth/login Login & get JWT public
POST /category Create a category auth required
POST /Query Raise a query auth required
PUT /Query/:id Edit your query auth required
DELETE /Query/:id Delete your query auth required
POST /Query/:id/comment Comment on a query public
POST /Query/:id/like Like a query public
